URL Encoder & Decoder

Quickly encode text into URL-safe format (percent-encoding) or decode URL strings back to plain text.

How to Use This Tool

To Encode (Make Text URL-Safe)

Use this when you need to safely include text in a URL. This process, also known as "percent-encoding," converts special characters (like spaces, !, #, ?) into a format that URLs can understand.

  1. Type or paste your plain text (e.g., Hello World!) into the 'Input' box.
  2. Click the 'Encode' button.
  3. The encoded result (e.g., Hello%20World!) will appear in the 'Output' box.

To Decode (Convert URL Code Back to Text)

Use this if you have an encoded URL string (e.g., Hello%20World!) and want to see the original, human-readable text.

  1. Paste your encoded string into the 'Input' box.
  2. Click the 'Decode' button.
  3. The original text (e.g., Hello World!) will appear in the 'Output' box.

You can use the 'Copy Result' button to copy the output to your clipboard and the 'Clear' button to reset both boxes.
